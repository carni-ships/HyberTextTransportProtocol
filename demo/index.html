HyberDB — On-Chain Database

HyberDB stores document patches as calldata on Berachain. No server, no SQL host — the chain is the database. Reads reconstruct state by replaying the patch chain from the contract head pointer. Requires HYBERDB_ADDRESS on the gateway.

Tableland — On-Chain SQL

Tableland is EVM-native SQL. Tables live on-chain; queries are served by Tableland's decentralized validator network. Injected as env.tableland with a simple query(sql) method.
